📌 Overview :

This repository contains a collection of C programs developed while learning and practicing core programming concepts. It serves as a personal reference as well as a helpful resource for beginners who want to understand C programming through examples.

The programs range from basic syntax and logic building to intermediate-level problem solving, covering a wide variety of topics.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🎯 Objectives :

Strengthen understanding of C programming fundamentals
Improve problem-solving and logical thinking skills
Build a strong foundation for system-level programming
Create a reusable reference for common programming patterns

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🧠 Topics Covered :

This repository includes programs based on:

🔹 Basic Syntax & Structure
🔹 Input / Output Operations
🔹 Conditional Statements (if, switch)
🔹 Loops (for, while, do-while)
🔹 Functions & Recursion
🔹 Arrays (1D & 2D)
🔹 Strings and String Manipulation
🔹 Pointers
🔹 Structures & Unions
🔹 File Handling
🔹 Dynamic Memory Allocation

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

⚙️ How to Run the Programs :

🛠️ Prerequisites :
C Compiler (e.g., GCC)
▶️ Steps

Clone the repository:

git clone https://github.com/PratikChavan19/C-Programs.git

Navigate to the folder:

cd C-Programs

Compile the program:

gcc program_name.c -o output

Run the executable:

./output
